Font-Face Rule

The @font-face rule allows you to define your own fonts. You can use this rule to include local fonts in your web pages. The @font-face rule has the following syntax:

@font-face {
    font-family: 'MyFont';
    src: url('myfont.ttf');
    font-weight: normal;
    font-style: normal;
    }

Colors and background color

You can use the style attribute in HTML to set the color and background color of an HTML element.

  • The color attribute is the color of the foreground
  • The background-color is the color of the background

Text-align

The text-align property can be used to align the text inside an element. The possible values for text-align are:

    Line-Heigt

    The line-height property is used to set the height of the line box.

    • A value of 1.5 means the line box is 1.5 times the height of a standard font size
    • A value of 1 means the line box is the height of a standard font size
    • A value of 2 means the line box is 2 times the height of a standard font size
