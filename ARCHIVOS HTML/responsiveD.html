Font-Face Rule

The @font-face rule allows you to define your own fonts. You can use this rule to include local fonts in your web pages. The @font-face rule has the following syntax:

@font-face {
    font-family: 'MyFont';
    src: url('myfont.ttf');
    font-weight: normal;
    font-style: normal;
    }

      Applying the style

      To apply your styles to your HTML elements, you can use the style attribute in HTML. The style attribute is a string that contains CSS property-value pairs. You can apply styles to individual HTML elements or to all elements on a page.
